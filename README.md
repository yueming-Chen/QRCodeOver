# QRCodeOver
##使用說明
1.安裝node.js
[node.js官網](https://nodejs.org/en/)
* 進入網站後找到Download
* 如果是windows就選擇該系統，linux及MacOS則選擇該系統
* 下載完成後，雙擊安裝

2.安裝完畢後，輸入下列指令

```
$ npm install -g express
```

3.安裝完express套件後，把下載下來的程式使用命令提示字元視窗進入到資料夾李，例如放在我的文件中

```
$ cd documents\QRCodeOver
```

4.進入後，輸入下面指令，並且到瀏覽器上輸入 `localhost:3000` 即可

```
$ node http.js
```
